# ClincalMind
ClinicalMind is an AI-powered medical chatbot that answers health-related questions using a Retrieval-Augmented Generation (RAG) pipeline. It ingests medical PDFs, indexes them into a Pinecone vector store using HuggingFace embeddings, and delivers concise, context-grounded responses via OpenAI's GPT-4o all through an intuitive Flask web interface.
